## 项目介绍
AppStore 地址
https://itunes.apple.com/us/app/煎蛋-有趣/id1230079100?l=zh&ls=1&mt=8

Github地址
https://github.com/totzcc/WeexJandan

采用Weex 做的iOS煎蛋客户端，
iOS拓展解析HTML 的基础能力。所有的业务逻辑代码都用Weex 实现
Android 客户端待研发
主要功能有
  - 文章
    - 热门文章
    - 文章分类
    - 文章详情
    - 收藏、分享文章
    - 评论列表、发表评论、评论点赞
  - 段子
    - 内容抓取
    - 点赞、踩
  - 无聊图、妹子图
    - 内容抓取
    - 多图同屏浏览

## 运行Weex项目

```
cnpm install
npm run dev
```

## 运行iOS项目 platforms/ios

```
pod install
```
